# Code Editor Setup

Visual Studio Code is a lightweight but powerful source code editor. While we recommend using Visual Studio for C#/C++ development, VS Code is excellent for Lua development in FiveM.

## Download

You can download Visual Studio Code from their [official website](https://code.visualstudio.com/).

## Essential Extensions

### Lua Development

- [Lua](https://marketplace.visualstudio.com/items?itemName=sumneko.lua)

  - Provides core Lua language support
  - Syntax highlighting
  - Code completion

- [CfxLua (FiveM/RedM) IntelliSense](https://marketplace.visualstudio.com/items?itemName=overextended.cfxlua-vscode)
  - Provides IntelliSense, diagnostics and annotations via Lua Language Server
  - Support for FiveM/RedM natives and runtime globals
  - Support for FiveM's Lua implementation

## Optional Extensions

Here are some additional extensions you might find helpful:

### Visual Customization

- [Bearded Icons](https://marketplace.visualstudio.com/items?itemName=BeardedBear.beardedicons)

  - Beautiful file icons
  - Helps you organize your files better

- [Nord Deep Theme](https://marketplace.visualstudio.com/items?itemName=marlosirapuan.nord-deep)
  - A nice theme which is easy on the eyes!
  - Personal pick from the Team

You can browse many more themes and icon packs on the [VS Code Marketplace](https://marketplace.visualstudio.com/vscode)

### Development Tools

- [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)
  - Enhanced Git integration
  - Use git from within VS Code
  - Requires [Git](https://git-scm.com/downloads) installation
