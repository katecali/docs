#### Dependencies
[ox_lib](https://github.com/overextended/ox_lib/releases) - Required to run this resource.<br/>
[oxmysql](https://github.com/overextended/oxmysql/releases) - **OPTIONAL** - Required if you want to run our custom code sharing system allowing players on your server to easily share outfits & vehicle configurations.
